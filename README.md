사용기술

* JavaScript(ES6+)
* React
* Redux, Redux-thunk
* MongoDB + Mongoose
* Node.js
* Express (Next.js 학습 예정)

구현 사항
1. 회원가입/로그인/로그아웃
2. Navigation Bar (사이드, 위쪽 고정)
3. Auth hoc
4. 제품 리스트 출력
4-1. 제품 리스트 페이지네이션
5. 제품별 상세 페이지 (사이즈 조견표, 제품별 내용 넣기 구현 필요)

1. SCSS, GSAP
2. 제품 이미지 커서 올렸을 때 줌 
3. 제품별 Q&A, Review(페이징 필요)
4. 장바구니
5. 좋아요 버튼
6. Notice (CRUD, 페이징)
7. 관리자 페이지
